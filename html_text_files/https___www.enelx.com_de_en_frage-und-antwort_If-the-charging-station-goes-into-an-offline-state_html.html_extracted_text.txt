If the charging station goes into an offline state, does it operate in a fail-safe mode?
If data connectivity is lost the JuiceBox will not enter a fail-safe mode, the JuiceBox will continue to operate/charge at the selected current value previously set within JuiceNet dashboard or the JuicePass App.
