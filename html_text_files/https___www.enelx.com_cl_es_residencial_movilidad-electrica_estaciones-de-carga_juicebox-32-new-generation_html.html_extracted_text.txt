Ficha Técnica
¡Carga tu vehículo en casa de forma rápida, fácil e inteligente!
Con tecnología de punta, el JuiceBox está equipado con toda la seguridad e inteligencia de control necesarias para hacer de la carga en casa la experiencia definitiva.
El JuiceBox permite el control directo tanto desde el usuario como desde sistemas de optimización de red inteligente, todo a través de nuestra plataforma de software de carga EV JuiceNet disponible para Android e iOS.
Fácil de usar
Interfaz de usuario para apoyar al cliente durante cualquier sesión de carga.
Velocidad de carga
Carga rápida con hasta 22 kW (AC), compatible con todos los vehículos eléctricos disponibles en el mercado: Socket Tipo 2 o Tipo 3a.
Seguro y confiable
Está equipado con sistemas de bloqueo que evitan la desconexión no autorizada de los enchufes durante la carga.

Login
Ingresa tu usuario y contraseña, y selecciona "Recuérdame" para acceder de manera más fácil a Enel X JuicePass.
También puedes acceder a la aplicación de Enel X utilizando Touch ID o Face ID ** Solo para teléfonos con esta opción
Mapa
Busca la estación de carga más cercana a tu ubicación en el mapa.
La red Enel X está creciendo todo el tiempo. Nuestras estaciones se pueden encontrar en hoteles, supermercados, centros comerciales y se identifican con un símbolo especial en el pin.
El color indica el estado de la estación de carga: verde-disponible, rojo-en uso, gris-en mantenimiento.
Estaciones de Recarga
Abre la sección de detalles de la estación para obtener la información más importante: dirección, número de serie, accesibilidad, potencia máxima y distancia.
Elije el mejor enchufe para tu vehículo para ver el precio establecido.
¿Tienes problemas? Presiona el botón Asistencia para informarnos sobre cualquier problema en la estación de carga y recibir asistencia.
Una vez que se haya seleccionado el enchufe, simplemente inicia la carga presionando "Cargar".
Reserva la estación que prefieras con el servicio de reserva.
