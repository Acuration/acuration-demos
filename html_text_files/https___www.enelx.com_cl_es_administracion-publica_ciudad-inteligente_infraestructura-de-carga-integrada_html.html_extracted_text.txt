Gamma Juice para la ciudad
Prestando una atención constante a la eficiencia y la optimización de los recursos, Enel X propone
un nuevo conjunto de soluciones integradas, que ofrecen múltiples servicios usando la misma base
infraestructural. Todo ello priorizando la digitalización y el diseño para hacer que el paisaje urbano sea
cada vez más moderno.
Sostenibilidad
Racionalizar los recursos disponibles, integrando varios servicios en una misma base infraestructural.
Seguridad

Usuarios
Comunicar mejor con el ciudadano, incluyendo pantallas digitales en los objetos presentes en el área urbana.
Recopilar datos
Recopilar datos para activar y optimizar servicios como la movilidad, el turismo y la seguridad.
