¿En que consiste el modelo PPA?
Es un acuerdo de compraventa de energía limpia a largo plazo desde un activo concreto y a un precio acordado entre un productor renovable y un consumidor. A través de un PPA, el cliente compra la energía producida a un precio reducido, en lugar de tener que hacer una gran inversión para financiar el proyecto fotovoltaico, es por esto que es una gran alternativa cuando no se posee la inversión necesaria para este tipo de proyectos ni los conocimientos técnicos para su mantención.
