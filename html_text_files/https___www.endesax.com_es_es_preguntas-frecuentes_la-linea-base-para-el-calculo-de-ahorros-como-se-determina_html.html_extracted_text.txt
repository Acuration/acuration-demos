¿La línea base para el cálculo de ahorros, cómo se determina?
Para la determinación de la línea base para el cálculo de ahorros se utiliza la funcionalidad del SGE creada para tal fin. Se utilizan históricos del contador de compañía si el consumo de HVAC respecto al consumo general es significativo. En caso contrario una vez instalados analizadores de red en el consumo de HVAC y tras un tiempo prudencial se determinaría la línea base para climatización.
