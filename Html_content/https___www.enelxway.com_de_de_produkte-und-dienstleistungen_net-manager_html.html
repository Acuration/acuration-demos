Net Manager

Das dynamische Dashboard für die totale Kontrolle über Ihre Elektromobilität.

Worum handelt es sich hierbei?

Net Manager ist eine intelligente Plattform, die für die Verwaltung von E-Fahrzeugflotten entwickelt wurde. Sie bietet einen vollständigen Überblick über Ihre privaten Ladestationen, die Verwaltung von Benutzerberechtigungen und die Daten der Ladevorgänge. Das Dashboard kann für die Verwaltung von Ladestationen am Arbeitsplatz oder für das Laden zu Hause durch die Mitarbeiter der Flotte verwendet werden. Es ist mit Waybox, Waypole und Waypump kompatibel.

Überwachung

Überwachen Sie alle Ladevorgänge. Erhalten Sie Zugriff auf vollständige Daten für jede Ladestation.

Ladeoptimierung

Optimieren Sie die vor Ort verfügbare Leistung für das Laden durch Lastausgleich zwischen simultanen Vorgängen.

Steuerung

Autorisieren Sie den Zugang von EV-Nutzern über die Enel X Way App.

Wie funktioniert es?

  1. Ladevorgänge
  2. Fahrer
  3. Karten
  4. Ladestationen
  5. Ladeoptimierung Plus
Das Dashboard bietet einen Überblick über alle von der Arbeitsplatz oder von Zuhause gesteuerten Ladevorgänge mit detaillierten Daten des einzelnen Vorgangs. Durch die Filterung können Sie benutzerdefinierte Berichte für eine umfassende Datenverwaltung herunterladen.
Wählen Sie im Abschnitt „Driver“ aus, wer Zugriff auf Ihren Dienst haben soll. Auf dieser Seite können Sie die Fahrerbenutzer in der Enel X Way App konfigurieren, aktivieren, bei Bedarf sperren und den vollständigen Gebührenverlauf einsehen.
Die RFID-Karte ist das ideale Werkzeug zur Verwaltung Ihrer Elektrofahrzeugflotte. Jede Karte kann auf dieser Seite verwaltet werden oder einem bestimmten Fahrer - Mitarbeiter – zugewiesen werden. Dieser kann die Karte mit seinem Firmencode in der Enel X Way App verbinden. Im Bereich „Karten“ besteht die Möglichkeit, die erworbenen RFID-Karten zu aktivieren oder (momentan oder permanent) zu sperren.

Jede gekaufte Waybox, jeder Waypole und jede Waypump kann auf dem Net Manager für Privatzahler konfiguriert werden, wobei Nutzung und Status ständig überwacht werden.

 

Fügen Sie neue Standorte außerhalb Ihres Hauptstandortes hinzu und ändern Sie diese beliebig. Ihr Flottenmanager kann die Ladeinfrastrukturen auch einem Standort zuweisen und einen lokalen Manager bestimmen.

Unsere Ladeoptimierungs-Software maximiert die Fahrzeugaufladung innerhalb der an Ihrem Standort verfügbaren Stromlast, indem sie die verfügbare Stromlast auf mehrere Ladestationen verteilt.

Die wichtigsten Vorteile

Mehrwert durch Net Manager

Mit der intelligenten Plattform des Net Manager können Sie die E-Fahzeugflotte Ihres Unternehemens managen, die Ladevorgänge Ihrer Mitarbeiter an firmeneigenen Ladestationen überwachen, Fahrer und Kartennutzer verwalten und den Status der eigenen Ladepunkte überprüfen.
