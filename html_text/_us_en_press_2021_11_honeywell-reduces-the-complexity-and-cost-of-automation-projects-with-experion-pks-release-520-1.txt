release 5201 honeywell reduces the complexity and cost of automation projects with experion pks release 5201 solution incorporates new features and functionality user experience enhancements and windows support november 10 2021 houston november 10 2021 honeywell today announced the latest release of its experion process knowledge system pks delivering new process automation functionality for end users experion pks release 5201 r5201 lays the foundation for experion pks highly integrated virtual environment hive honeywells nextgeneration control technology experion pks hive enables automation projects to be deployed in a more flexible and resilient manner by decoupling control system elements that are traditionally engineered configured and deployed in a hierarchal manner in turn the standardization and optimization of projects is enabled while simplifying lifecycle and support management the modular flexibility of the r5201 allows experion pks hive to scale to fit any size of automation project this can reduce the time cost and impact of late changes associated with project design delivery and implementation the r5201 solution can be implemented seamlessly and incrementally across a customers current installed base leveraging existing investments said jason urso chief technology officer honeywell process solutions it delivers higher levels of system resiliency and availability while reducing the overall cost of automation projects in addition experion pks r5201 introduces several experion batch features including the capability to host batch history directly on the experion server and a new reporting tool that allows process engineers to easily create and manage reports the solution also features electronic work instructions that allow users to digitize processes reduce operator entry error increase data integrity and feed into the electronic batch record r5201 also introduces the updated version of the c300 controller with several upgrades controledge unit operations controller uoc enhancements include simplified processes to create complex control strategies with custom algorithm blocks that run directly on the uoc and opc unified architecture connectivity to devices directly on the uoc there are also numerous user experience improvements that increase operator responsiveness and simplify workflows r5201 can be applied to end users spanning the oil and gas refining petrochemical power generation life sciences pulp and paper mining minerals and metals specialty chemicals and renewable energy sectors to learn more
