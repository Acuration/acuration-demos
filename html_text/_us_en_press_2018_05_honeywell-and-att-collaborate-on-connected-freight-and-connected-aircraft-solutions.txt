pressure the sensors are interconnected using a mesh network protocol which enables them to efficiently communicate data back to a nearby centralized gateway device logistics providers access a dashboard to receive status alerts and can react quickly to incidents that occur during shipment to prevent damage and loss with the connected aircraft honeywell is using the att global network to
