eddi has the possibility of installing a relay  sensor card which allows for many different wiring configurations and includes the following functions


 two independent multifunction relays 16amp

 esense input isolated 230v detection for offpeak hours etc

 two temperature sensors pt1000


the two relays can be set independently for different functions or they can be set to work as a pair relay functions include


 export threshold

 import threshold value

 use during heating

 legionella control

 timed actions

 fault signal for bms systems
