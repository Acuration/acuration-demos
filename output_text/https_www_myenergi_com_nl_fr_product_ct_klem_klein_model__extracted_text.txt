pince ct petit modèle


zappi  eddi sont fournis avec un terminal ct câblé en standard


le zappi triphasé de 22kw est livré en standard avec 3 tc


une ou 3 dans le cas dun onduleur triphasé pince ct supplémentaire est nécessaire si vous voulez voir la sortie en temps réel de votre système solaireéolien sur lécran lcd du produit


il sagit dune version plus petite de la pince ct standard vous pouvez utiliser cette ct pour mesurer jusquà 40a de plus il ne sagit pas dun modèle clipon il faut donc détacher le câble correspondant pour le faire passer dans le ct cest seulement pour lélectricien avancé 


avec cette borne à pince zappi peut prendre des mesures sur les câbles de phase du fusible principal pour sa protection la consommation de la maison le surplus de votre propre production mais aussi pour le rendement de votre système énergétique panneaux solaires ou éolienne


avec le zappi monophasé et eddi 1 ct est fourni en standard avec le zappi triphasé 3 pinces ct sont fournies


avec un tc supplémentaire vous pouvez également afficher le rendement de vos panneaux solaires et le lire sur lécran il nest pas nécessaire pour un fonctionnement correct de placer également un tc sur londuleur cela nest nécessaire que pour obtenir un aperçu complet sur lécran et via le hub dans lapplication de toutes les informations sur la provenance ou la destination de lélectricité


pour les modèles zappi v1  si vous voulez faire une mesure triphasée vous devez utiliser 3 tc en combinaison avec le harvi  vous ne pouvez connecter que 2 bornes câblées dans le zappi v1


pour les modèles zappi v2  si vous voulez faire une mesure triphasée vous pouvez connecter 3 bornes directement filaire etou connecter 3 bornes supplémentaires sans fil avec un harvi
