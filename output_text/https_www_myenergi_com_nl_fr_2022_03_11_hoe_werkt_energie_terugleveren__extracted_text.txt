comment fonctionne le rachat dénergie 
 


 








 










grâce aux panneaux solaires vous pouvez produire votre propre énergie vous pouvez utiliser cette énergie vousmême directement mais parfois vous générez plus dénergie que vous nen utilisez ce surplus dénergie est réinjecté dans le réseau nous appelons cela réinjecter de lénergie à lheure actuelle vous recevez encore une compensation pour le renvoi dénergie mais celleci est progressivement supprimée comment fonctionne exactement ce système dalimentation 


comment fonctionne le rachat dénergie 


lorsque vous faites installer des panneaux solaires ils doivent être raccordés au réseau vous devez informer votre gestionnaire de réseau que vous allez réinjecter de lélectricité si nécessaire le gestionnaire de réseau peut se rendre chez vous pour installer un compteur dalimentation adapté en outre de nombreux gestionnaires de réseau vous demanderont de vous inscrire pour un rachat dénergie via 
energieleverennl
 la procédure exacte denregistrement varie dun gestionnaire de réseau à lautre vérifiez donc auprès de votre propre gestionnaire


questce que le programme déconomie dénergie 


si vous réinjectez de lénergie vous recevez actuellement une 
compensation financière
 pour cela  cela se fait par le biais de ce que lon appelle le système de comptage net toutefois 
à partir de 2024 le dispositif de comptage net disparaîtra
 très probablement vous recevrez alors un remboursement de plus en plus faible pour lénergie que vous réinjecterez il est donc financièrement plus intéressant dutiliser autant que possible lénergie que lon produit soimême vous ne payez rien pour lutilisation de votre propre énergie solaire si vos panneaux solaires ne produisent pas assez dénergie vous payez lénergie que vous utilisez à partir du réseau il est donc utile de faire preuve dintelligence avec son énergie en outre le réseau électrique sera moins sollicité lorsque davantage de personnes produiront et consommeront ellesmêmes de lénergie


comment puisje utiliser mon énergie de manière judicieuse 


lorsque le retour de lénergie nest plus financièrement avantageux il est préférable dutiliser sa propre énergie lénergie que vous utilisez immédiatement ne vous est pas restituée par exemple lorsque vous faites fonctionner votre machine à laver tout en produisant de lénergie solaire vous utilisez cette énergie immédiatement pour être efficace avec votre énergie il est donc judicieux dutiliser les appareils électriques autant que possible lorsque le soleil brille

en utilisant nos produits intelligents dénergie verte vous pouvez tirer le meilleur parti de votre propre énergie grâce à zappi vous pouvez recharger votre voiture électrique avec votre propre énergie solaire si vous combinez ce chargeur ev avec notre eddi innovant vous pouvez également chauffer votre eau
